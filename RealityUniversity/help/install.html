There is really nothing to "install" in the usual sense. The entire application resides in one folder. That folder contains three items:

  • a "help" folder which contains these help files in .html form.
  • the database file with a .dat file extension
  • the java executable in the form of a .jar file

A screenshot with a list of them in a typical windows explorer view is shown below.

The "installation" procedure merely involves dragging and dropping the folder containing the above items into wherever you usually have your programs installed. Usually this is inside the "C:\Program Files" folder. And yes, it's that simple!

The only complication here really is putting a link to the java executable in the program listing that shows up whenever you click the Start button (if you can manage to have it show up in Windows 8.0 or 8.1!) and click the "All Programs" link that shows up.
