Installation Procedure

The RealityU Surveyor Application is inside a zipped folder (RealityU Surveyor Setup.zip)
that you can get from a flash drive or a CD.

  1. First you download the zip folder onto your computer
  2. Next, you open up the zip folder and click on the application file
  3. Click Run
  4. A Setup Wizard pops up and you just follow through the steps for this:
    • Process for Setup Wizard:
    • Next, then Accept License Agreement
    • Select Destination Location
    • Select Menu Folder
    • Select Additional Tasks
    • Ready to Install(?) Click on Install
  5. Once that is done, a desktop icon will appear
  6. Click on it, and the application will boot up on the desktop
