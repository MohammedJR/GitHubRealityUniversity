Processing: How It's Done

The user simply presses the 'Save and Process Surveys' button
in the Survey Form page to start up the processing.

Processing is done within the application, so there's no need
for survey volunteers to make judgement calls on the survey data.
This helps to expedite the data entry process as manual data entry
and processing is very time and labor intensive.

Data Storage: The Database

Student Data is stored in an embedded database also within the application.
The application allows admins to go in and make changes to it via GUI interfaces.
For example, if one wishes to add/delete jobs and related information, then
one can certainly do that within the Settings panel.

In case the database gets deleted, which can happen, the application when opening up
will check for all data tables. If it finds one missing, then it will recreate it
from a backup file.
